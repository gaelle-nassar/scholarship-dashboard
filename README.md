# Scholarship Dashboard

## How to Run
```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Experience with Claude Code

Claude Code made it significantly easier to set up and build this project. Since Nuxt 3 was new to me, I relied on Claude mainly for syntax help and to help me build features in the best way, optimized for Nuxt. It helped me understand specific concepts in Nuxt, like auto-imports and composables. I made sure that I read,  understood, and tested every piece of code it produced before moving on.
